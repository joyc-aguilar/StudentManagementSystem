# Student Management System

A JavaFX-based application designed to manage student records, featuring full CRUD (Create, Read, Update, Delete) functionality with a PostgreSQL database.

## Features
- **Add Students:** Easily register new students with their name, course, and year level.
- **View Records:** Displays all student data in an organized table format.
- **Update/Delete:** Allows for easy modification or removal of student records.
- **Database Integrated:** Uses PostgreSQL for persistent data storage.

## Prerequisites
- **Java 21** or higher
- **Maven**
- **PostgreSQL** database

## How to Run
1. Clone the repository:
   `git clone https://github.com/joyc-aguilar/StudentManagementSystem.git`
2. Open the project in IntelliJ IDEA.
3. Ensure your PostgreSQL database is running and the connection details match your `DBConnection.java` file.
4. Use the Maven panel in IntelliJ to run the project:
   - Navigate to `Plugins` -> `javafx` -> `javafx:run`
   - Double-click `javafx:run` to launch the application.
